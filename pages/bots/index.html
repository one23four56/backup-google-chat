$RoomName$ Bot List

This list contains all the bots allowed in $RoomName$. The owner of $RoomName$ can modify this list.

To use a bot, type one of the commands listed in its commands section.

  • Some commands (like /8ball) have no arguments, to use those you just type the command.
  • Other commands (like /roll) have arguments, to use those commands you have to type the command along with the arguments.
    • For arguments surrounded in [brackets], you just have to type the argument (i.e. /roll 1)
    • For arguments surrounded in 'quotes', you have to type the argument in quotes (i.e. /stats 'my name')
    • Arguments that end in a question mark (?) are optional, you do not have to include them.
    • For commands with multiple arguments, be sure to separate the arguments with spaces.
