How to Use Commands:
If a command is listed in the commands list on its own (no arguments), simply type the command in the message box to use it.
If the command has arguments, type the command and arguments in the message box. With arguments, however, please note that:

  • If an argument name listed here is surrounded by brackets (i.e. [number]), it is one word, and can be typed without the brackets (i.e. /roll 1)
  • If an argument name listed here is surrounded by quotes (i.e. 'name'), it is multiple words and MUST be typed WITH the quotes (i.e. /stats 'Example User')
  • If an argument name listed here is followed by a '?' (i.e. 'name'?), it is optional, and can be omitted
